## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: Node.js

### Setup

```
  RUN mkdir my-folder
  RUN cd my-folder
  RUN git clone https://github.com/fezzopro/morse-code-decoder.git
  RUN cd morse-code-decoder

```

### Install


```
  RUN bundle install

```

### Usage

 - Install Ruby to your environment
 - **RUN** `ruby morse_code_decoder.rb`
